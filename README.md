# AmidaGo! - プレミアムあみだくじアプリ

モダンなデザインとスムーズなアニメーションを備えた、HTML5/CSS/Canvasベースのあみだくじアプリケーションです。

## 特徴

- **モダンなUI/UX**: ダークモードを基調とした、ガラスモルフィズムデザイン。
- **Canvasアニメーション**: 経路をなぞるスムーズなアニメーション表示。
- **動的レイアウト**: 2名から20名まで、参加人数に応じた柔軟なスケーリング。
- **完全なランダム性**: 毎回異なるあみだの生成と、内部的なゴールマッピングのシャッフル。
- **動的な結果表示**: 経路を完走するまで結果がわからない演出。

## 使い方

1. `index.html` をブラウザで開きます。
2. **参加人数**（2〜20名）を入力します。
3. 必要に応じて**ゴール名**（初期値は1, 2...）を編集します。
4. 「**あみだを生成**」ボタンをクリックしてあみだクジを作成します。
5. 上部の **A, B, C...** ボタンをクリックするとアニメーションが開始されます。
6. 終点に到達するとゴール名が表示され、下部の結果エリアに確定した結果が表示されます。

## 技術スタック

- **Structure**: HTML5
- **Style**: CSS3 (Vanilla CSS, Glassmorphism)
- **Logic/Animation**: JavaScript (Canvas API, requestAnimationFrame)
